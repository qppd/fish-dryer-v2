# Glossary

| Term | Definition |
|------|------------|
| **BMS** | Battery Management System — protects LiFePO4 battery from overcharge/discharge |
| **CRC** | Cyclic Redundancy Check — error-detecting code (used by SHT31 with poly 0x31) |
| **EDT** | Estimated Drying Time — calculated from water loss rate over 10-minute windows |
| **EEPROM** | Electrically Erasable Programmable Read-Only Memory — used for calibration persistence on Nano |
| **ESP-Now** | Espressif's connectionless wireless protocol for direct device-to-device communication |
| **HMI** | Human-Machine Interface — the touchscreen display and its software |
| **HX711** | 24-bit delta-sigma ADC for load cell weight measurement |
| **I2C** | Inter-Integrated Circuit — serial bus used by SHT31 (and legacy protocol) |
| **IO Expander** | CH422G — provides additional GPIO pins via I2C |
| **LiFePO4** | Lithium Iron Phosphate — safe, long-life battery chemistry |
| **LVD** | Low Voltage Disconnect — prevents deep battery discharge |
| **LVGL** | Light and Versatile Graphics Library — embedded GUI framework |
| **MPPT** | Maximum Power Point Tracking — efficient solar charge controller |
| **PID** | Proportional-Integral-Derivative controller — closed-loop temperature regulation |
| **PSRAM** | Pseudo-Static RAM — external memory on ESP32-S3 for large display buffers |
| **RGB565** | 16-bit color format (5 bits red, 6 bits green, 5 bits blue) |
| **SHT31** | Sensirion temperature and humidity sensor (±0.3°C, ±2% RH) |
| **SoftwareSerial** | Arduino library for serial communication on any digital pins |
| **SoftwareWire** | Software I2C implementation (used by Nano for SHT31) |
| **SSR** | Solid State Relay — electronically switched relay for heater/fan/exhaust control |

| NaN | Not a Number (IEEE 754) |
| PCLK | Pixel Clock (display signal) |
| RH | Relative Humidity |
| SCK | Serial Clock |
| SCL | Serial Clock Line (I2C) |
| SDA | Serial Data Line (I2C) |
| STA | Station mode (Wi-Fi) |
| VSYNC | Vertical Synchronization (display signal) |
| XOR | Exclusive OR (checksum algorithm) |
